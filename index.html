Credibility Score AI

Credibility Analysis Visual

The Challenge: Addressing the need to reliably assess the credibility of information sources, especially in automated systems like RAG-based chatbots.

My Solution: Developed a proof-of-concept system to generate a credibility score for articles and references. This involved exploring Machine Learning, Rule-Based, and Hybrid approaches to analyze various textual and metadata features.

Key Features & My Role:

  • Researched and prototyped different scoring algorithms based on source authority, content analysis, and citation patterns.
  • Designed a framework for integrating this scoring mechanism into a Retrieval-Augmented Generation (RAG) pipeline.
  • Focused on identifying features indicative of bias, trustworthiness, and reference quality using NLP techniques.

Technologies Used: Python, Jupyter Notebooks, Natural Language Processing (NLP) libraries (e.g., NLTK, spaCy), RAG principles, Scikit-learn.

DataScience-Agent

DS-Agent Workflow

The Challenge: Simplifying complex data science workflows and making advanced analytics accessible to a broader audience through natural language interaction.

My Solution: This project focuses on building an intelligent agent capable of understanding natural language queries to perform various data science tasks such as data loading, cleaning, analysis, visualization, and basic model training.

Key Features & My Role:

  • Built an interactive user interface using Streamlit for easy engagement with the agent.
  • Utilized Hugging Face models for specific NLP tasks and embedding generation.

Technologies Used: Python, Streamlit, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, LLMs (Mistral/OpenAI API), Hugging Face Transformers.
